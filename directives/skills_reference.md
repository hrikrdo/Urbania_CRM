# Directive: Skills Reference

## Objetivo
Documentar las habilidades disponibles para el agente y cómo utilizarlas.

## Ubicación
Las skills están instaladas en `.claude/skills/` (237 skills disponibles)

## Cómo Usar Skills
Menciona la skill con `@` o `/` en el chat:
```
@brainstorming ayúdame a diseñar el CRM
/stripe-integration agregar pagos
```
